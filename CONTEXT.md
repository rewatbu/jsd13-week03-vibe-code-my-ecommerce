# Full Stack project on a local computer

### Tools details
- Frontend with HTML, CSS, and JavaScript.
- Not to use React.js for this project
- Backend with Node.js and Express.js
- Database ong MongoDB Atlas, schema models have been provided in the root directory

### Business context
- read the my-ecommerce-business.md file

### Technical requirements
- use /apps folder for backend
- use /web folder for frontend
- Web pages language should be in Thai language, only use English for necessary terms for better user experience
- add images that could be used from the internet
- use simple style for the web design, premium feeling but price friendly
